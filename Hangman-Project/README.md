# 😵 Classic Hangman Terminal Game

A clean and interactive implementation of the traditional Hangman word-guessing game, written in **Python**.

## ✨ Features
- **Extensive Vocabulary:** A built-in library of over 200 words across various categories.
- **ASCII Art Animation:** Dynamic rendering of the "Hangman" stages based on remaining tries.
- **Input Validation:** The game smarty handles duplicate guesses and keeps track of your progress.
- **Terminal-Based Experience:** Simple, lightweight, and runs directly in your command line.

## 🕹️ Gameplay
1. The game selects a random word.
2. You have **6 tries** to guess the letters.
3. For every wrong guess, the hangman drawing progresses.
4. Win by revealing the full word before the hangman is complete!

## 🚀 Getting Started
Simply run the script using Python:
`python Hangman.py`
